# Land Scan Apis
> Buid using NodeJS, Express, MongoDB and Mongoose

## Assignment

	[link to assignment](/assignment.pdf)

### Prerequisites

#### Node.js

> Version: >= 10

#### MongoDB

> Version: >= 4

#### Setup

	cd land_scan_api
## Install

	npm install

## Load Dev Data

	> To Import Dev data
	npm run import:devData

	> To Delete Dev data
	npm run delete:devData

## Start Up

	npm start
